Prerequisites:

Angular CLI
Node.js with NPM 

IDE used: Visual Studion Code

How to run application:
-Install Angular CLI and Node.js with NPM. 
-Import the source code to visual studio code.
-Run the command 'npm install' to install all the node modules.
-Run the command 'ng serve' to run the application.
-Go to the browser at 'http://localhost:4200/'
